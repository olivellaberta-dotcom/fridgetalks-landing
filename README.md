# 🍳 Fridgetalks

Landing page de Fridgetalks — Cocina con lo que tienes.

Una app que te ayuda a aprovechar los ingredientes que tienes en casa mediante reconocimiento por imagen.

## Stack
- React + Vite
- Tailwind CSS
- Lucide Icons

## Desarrollo local
```bash
npm install
npm run dev
```
